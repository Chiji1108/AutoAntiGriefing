# AutoAntiGriefing
開発したGriefing判定アルゴリズムとは
<p><実装済計算ブロック><br>
<br>
[GP]=+10×{<br>
×ブロックの種類<br>
×人工ブロックか自然ブロック<br>
×自分が置いたか置いてないか<br>
×プレイヤー新参度<br>
×ログインからの経過時間<br>
×ブロック連続破壊度<br>
×ブロック破壊効率度<br>
×ブロック破壊時被プレイヤー状態<br>
}<br>
<br>
GPが50000以上で荒らし判定<br>
